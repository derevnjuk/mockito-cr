Mocking library for Crystal inspired by [mockito](https://site.mockito.org/).

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     mockito-cr:
       github: derevnjuk/mockito-cr
   ```

2. Run `shards install`
